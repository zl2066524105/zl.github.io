通过用户查询菜单列表

接口地址 /system/cfg/menu

请求方式 GET

consumes ["application/json"]

请求参数

暂无

响应参数

参数名称 参数说明 类型 schema
component 组件 string
iconCls 图标 string
id id integer(int32) integer(int32)
name 菜单名 string
path path string
requireAuth 是否要求权限 boolean
roles 角色列表 array Role对象
url url string

Menu对象

参数名称 参数说明 类型 schema
component 组件 string
iconCls 图标 string
id id integer(int32)
name 菜单名 string
path path string
requireAuth 是否要求权限 boolean
roles 角色列表 array Role对象
url url string

Role对象

参数名称 参数说明 类型 schema
id id integer(int32)
name 名称 string
nameZh 角色名称 string
