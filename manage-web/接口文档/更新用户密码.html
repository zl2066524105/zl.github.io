更新用户密码

接口地址 /admin/pass

请求方式 PUT

consumes ["application/json"]

请求参数

参数名称 参数说明 请求类型 是否必须 数据类型 schema
oldPass 旧密码 body true object
pass 新密码 body true object
adminId id body true object

请求示例

{
	"adminId": ,
	"oldPass":"",
	"pass":""
}

响应参数

参数名称 参数说明 类型 schema
code integer(int64) integer(int64)
message string
obj object

响应示例

{
	"code": 0,
	"message": "",
	"obj": {}
}
